## QWeather Icons

The weather display uses selected icons from [QWeather Icons v1.8.0](https://github.com/qwd/Icons/tree/v1.8.0), created by [QWeather](https://www.qweather.com/).

The icon designs are licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). The source repository code is licensed under the MIT License.

Used icon codes: `100`, `150`, `103`, `153`, `104`, `305`, `306`, `307`, `310`, `400`, `401`, `402`, `501`, `502`, `504`, `2208`, and `999`.

Modifications: the SVG artwork is rasterized, inset into a 48 x 48 canvas, reduced to a monochrome 1-bit mask, and recolored at runtime to match each clock theme.
